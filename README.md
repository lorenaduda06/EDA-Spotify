# Análise Exploratória de Dados: Spotify Songs

O projeto tem o intuito de proporcionar uma análise detalhada de um dataset constituído por mais de 30.000 faixas do Spotify, de 1957 até 2020.
Essa análise explora a forma como as características técnicas podem influenciar a popularidade das faixas.

## Insights
* **Gênero mais popular:** Foi identificado que o gênero **Pop** apresenta a maior mediana de popularidade.
* **Evolução da métrica Dançabilidade:** A métrica apresentou crescimento constante a partir dos anos 80, e atingiu estabilidade após os anos 2000
* **Aspectos Técnicos:** Há uma correlação positiva direta entre as métricas de **intensidade** e **volume** das faixas.

## Tecnologias utilizadas:
* **Python** (Pandas e Numpy)
* **Gráficos:** Matplotlib, Seaborn e Plotly (interativo)
